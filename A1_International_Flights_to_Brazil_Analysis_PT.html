ANALISE DOS VOOS INTERNACIONAIS PARA O BRASIL¶

H muito tempo sem brincar com os dados, ento resolvi pesquisar e coletar na Internet alguns dados abertos para preencher o meu tempo e a minha cabea. Encontrei disponvel no Portal da ANAC estes dados de 2015, sobre a aviao comercial no Brasil. E rapidamente algumas perguntas/curiosidades vieram em minha mente:

  1. Qual o comportamento dos voos internacionais no Brasil no perodo de 1 ano?
  2. Quais so os Aeroportos (Top10) com um grande volume de voos para o Brasil?
  3. Quais so os Aeroportos (Botton10) com um pequeno volume de voos para o Brasil?
  4. Quais so os Aeroportos brasileiros (Top10) com grande chegada de voos internacionais?
  5. Quais so as CIAs AEREA que mais fazem voo para o Brasil?
  6. Durante os meses de maior trfego quem so os Pases com grande volume de voos para o Brasil?
  7. Durante os meses de maior trfego quais so os aeroportos brasileiros com grande chegada de voos internacionais?
  8. Durante os meses de maior trfego quais so as CIAs que mais fazem voo para o Brasil?
  9. Durante os meses de maior trfego este volume de voos internacionais para o Brasil representado por pessoas vindo conhecer o Pas ou brasileiros saindo para o exterior?
  10. Qual comportamento dos voos que chegam ou saem do Brasil, esto lotados ou vazios?

Bem, estas so algumas das perguntas/curiosidades que eu gostaria de obter respostas. Ento, vamos ao desafio de buscar respostas para estes questionamentos e alguns outros insights.
